Matemática

O que é um número primo?

R: Um número primo é um número maior que 1 que possui apenas dois divisores: 1 e ele mesmo.

Matemática

Como se calcula a área de um círculo?

R: A área de um círculo é calculada pela fórmula A = π x r², onde r é o raio.

Matemática

O que é uma função linear?

R: Uma função linear é uma função do tipo f(x) = ax + b, onde a é a inclinação da reta e b é o ponto onde a reta cruza o eixo y.

Matemática

Como resolver uma equação do 2º grau?

R: Use a fórmula de Bhaskara: x = (-b ± √(b² - 4ac)) / 2a, onde a, b e c são os coeficientes da equação ax² + bx + c = 0.
